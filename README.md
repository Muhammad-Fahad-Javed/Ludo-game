# 🎲 OOP Ludo Game

A console-based Ludo Game developed in **C++** using **Object-Oriented Programming (OOP)** principles. This project simulates the classic Ludo board game and demonstrates the practical implementation of classes, objects, encapsulation, inheritance, polymorphism, and modular programming.

---

## 📌 Project Overview

The OOP Ludo Game is designed to recreate the traditional Ludo experience in a C++ console environment. Players roll dice, move tokens across the board, and compete to reach the destination before their opponents.

The project was developed as part of an Object-Oriented Programming course to apply theoretical OOP concepts in a real-world software project.

## 📚 OOP Concepts Implemented

### Encapsulation
Data members and methods are grouped into classes.

### Abstraction
Complex game operations are hidden behind class methods.

### Inheritance
Derived classes extend functionality from base classes.

### Polymorphism
Functions can behave differently depending on context.

### Classes and Objects
Core game entities are represented using classes and objects.

## How to Play

1. Start the game.
3. Roll the dice on your turn.
4. Move tokens according to dice value.
5. Follow standard Ludo rules.
6. Reach the final destination before other players.
7. First player to finish wins.
